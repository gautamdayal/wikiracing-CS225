Detailed Description

Class to store the connections between wikipedia pages as a graph data structure.

Given a collection of csv files storing the source and destination wikipedia pages for a single page, the class encapsulates functionality to parse this data, store it as an adjacency list, and compute the shortest path between any two pages that are in the stored subgraph. Functionality to compute strongly connected components is also included.
